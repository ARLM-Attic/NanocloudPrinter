#Nanocloud Printer

This project is a remote desktop printer made in C++. It relies on Ghostscript, which is an interpreter for the PostScript langage and RedMon, a redirection port monitor.

Your file tree should be something like:
```
├── CCPSRendering
├── Common
├── General
│   └── res
├── Install
│   ├── DriverFiles
│   ├── DriverFiles64bit
│   ├── Images
│   ├── lib
│   └── urwfonts
├── NanocloudPrinter
├── ghostscript
├── installDll
│   └── Debug
└── redmon
```
